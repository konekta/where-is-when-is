    Konekta is a web service that supports the collection, validation and access to community based services.

    About the project

    We are focused on three major user groups:

    A mobile user looking for services nearby.

    Example user story:

    'As a young mother I would like to know which community services are available in my area.'

    The service is made available through our mobile client.

    The development practitioner interested in sharing and validating locations.

    Example user story:

    'As a development practitioner I would like to share and validate open data so that others find the service.'

    The service is made available through the website location upload form and publication to an open database.

    The developer interested in developing applications that use this data.

    Example user story:

    'As a developer I need access to reliable data so that I can build applications.'

    The service is made available through the JSON web service.

    Objectives

    • Deliver services to users on the ground.
    • Build relationships with organisations delivering community services on the ground
    • Definition of users and needs.
    • Establish a dedicated open data repository focused on community services.
    • Identification of useful open data.
    • Publication of valuable timely open data.
    • Engage open source developers.
    • Engage data journalists.
    • Establish a business model that powers the development of open data.
